## v3.0 (2026-07-02)

### ✨ 新增功能
- **👑 访客/管理员双模式** — 密码登录保护管理功能
  - 访客模式：只读浏览、搜索、复制、导出（无需认证）
  - 管理员模式：增删改、导入、云同步（需密码登录）
  - 首次访问自动进入访客模式，数据从 CDN 极速加载
- **🔐 密码认证系统** — Web Crypto API SHA-256 哈希
  - 首次使用创建管理员账号（用户名 + 密码 ≥ 6 位）
  - 凭据存储在 `localStorage`，每次登录验哈希
  - 登录/退出即时切换 UI 模式
- **⚡ CDN 公共读取** — 访客通过 `raw.githubusercontent.com` 加载云端数据
  - 无需 Token、无 CORS 预检、国内网络更稳定
  - 页面加载自动比对本地/CDN 数据版本

### 🎨 UI/UX 改进
- 登录弹窗：淡入居中，支持点击蒙层/ESC 关闭
- 顶部导航栏自适应模式显示：访客看 🔐 登录按钮 + 🔒 访客模式徽章；管理员看 👑 管理员 + 管理操作按钮 + 🚪 退出
- 所有管理按钮（添加/编辑/删除/导入/批量操作/同步）仅在管理员模式下可见
- `N` 快捷键仅在管理员模式下可用

### 🔧 内部优化
- `init()` 根据 `admin_logged_in` 分支加载：管理员走 API 同步拉取，访客走 CDN 公共拉取
- `updateModeUI()` 通过 `body.admin-mode` / `body.visitor-mode` CSS 类切换显示
- CSS 使用 `!important` 确保双模式覆盖优先级
- `fetchPublicData()` 从 raw CDN 读取，失败时返回 null 不阻塞页面
