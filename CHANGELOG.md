# 更新日志

## v2.0 (2026-07-02)

### ✨ 新增功能
- **收藏/星标** — 每条命令可收藏，支持筛选仅收藏项，收藏卡片有金色边框标识
- **排序选项** — 支持按使用频率、最近添加、名称排序
- **模糊搜索** — 多词空格分隔 + 字符级模糊匹配（subsequence match）
- **复制追踪** — 自动记录复制次数，卡片标签栏显示 `📋 N`
- **撤销删除** — 删除后 5 秒内可一键恢复
- **批量操作** — 复选框多选 + 批量删除 / 批量导出
- **多格式导出** — JSON / Markdown / CSV 三种导出格式
- **键盘快捷键面板** — 按 `?` 弹出快捷键说明
- **分页浏览** — 每页 30 条，带页码导航

### 🎨 UI/UX 改进
- **弹窗动画** — modal 淡入 + 缩放动画，快捷键面板同理
- **Toast 滑入** — 从右侧弹性滑入，上限 5 个防堆积
- **搜索框加图标** — 🔍 图标固定于输入框左侧
- **卡片悬停增强** — 渐变色 overlay + 更深阴影
- **命令代码复制提示** — hover 时右上角显示"📋 复制"浮层
- **分页按钮美化** — 活跃态渐变背景 + 阴影
- **统计加图标** — 📦 总命令 / 👁️ 当前显示
- **表单焦点发光** — 蓝色光晕 focus 态
- **自定义滚动条** — 暗色主题窄条滚动条
- **选中文本颜色** — `::selection` 天蓝色背景
- **平板/移动端适配** — 1024px + 768px 断点优化
- **底部留白** — 页面底部增加 padding

### 🐛 Bug 修复
- 修复 CSS `.toast` 空块导致 toast 容器样式失效
- 删除重复定义的 `updateSubCategoriesOptions` 函数
- 补全 `btn-secondary` 样式（弹窗取消按钮）
- `defaultCommands` 缺失 `favorite` / `copyCount` 字段（自动补齐）
- `saveCommand` 增加输入 trim + 空值校验
- `N` 快捷键增加弹窗状态检查，避免弹窗已开时重复触发
